# :green_heart: 서비스 소개
:point_right: 메인 페이지에서 가장 많이 팔린 상품을 우선순위로 보여주어 유저들이 인기있는 상품에 쉽게 접근할 수 있습니다.
  
:point_right: 상품페이지에서 유저들이 원하는 카테고리별로 편리하게 접근할 수있습니다.
  
:point_right: 상품 디테일 페이지에서 로그인을 한 유저들이 원하는 상품마다 댓글을 달고 해당하는 상품의 평을 보고 쉽게 접근할 수 있습니다.
  
:point_right: notice페이지에서 유저들이 홈페이지에 새롭게 업데이트된 소식을 실시간으로 전달 받을수있습니다.
  
:point_right: 돋보기 창을 눌러 페이지 내 어디서든 유저들이 원하는 상품을 검색할 수 있습니다.
  
:point_right: 하트 아이콘을 누르면 페이지 내 어디서든 위시리스트를 확인할 수 있습니다.
  
:point_right: mypage로 이동시 유저들이 주문한 내역, 배송, 회원정보 수정, 리뷰 관리를 쉽게 접근할 수 있습니다.
  
:point_right: 관리자 페이지와 유저 페이지를 따로 나누어 관리자의 역할을 수월히 가능하게 하였습니다.
  
:point_right: 관리자는 모든 게시물을 수정, 삭제, 등록할 수 있으며, 회원 관리뿐만아니라 회원에게 보낼 상품들의 배송관리 또한 맡고 있습니다.
